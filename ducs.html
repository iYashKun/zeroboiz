Captcha API

Secure, fast, and production-ready CAPTCHA service with enterprise-grade security.

v2.0 • Production Ready

Overview

A modern CAPTCHA solution built for developers who need maximum security with minimal friction.

  • JWT-based authentication with IP locking
  • 3-attempt limit per token
  • Secure HttpOnly cookies
  • Full security headers
  • AWS Lambda ready

Endpoints

GET

/captcha.rb — Generate CAPTCHA

Creates a new CAPTCHA challenge and returns tokens + image data.

GET

/captcha.rb?png=TOKEN — View Image

GET

/captcha.rb?verify=TOKEN&captcha=ANSWER — Verify

Code Examples

JavaScript (Recommended)

async function generateCaptcha() {
  const res = await fetch('/captcha.rb');
  const data = await res.json();
  document.getElementById('captcha-image').src = data.data;
}

Security First

Built with defense-in-depth. Every request is validated against IP, expiry, and attempt limits.
