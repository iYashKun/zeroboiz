micx browser microphone API docs

micx gives you a polished way to inspect live microphone health, voice activity, and spectral data from the browser.

Getting started

Import the module, request a stream, and create the API wrapper.

import MicrophoneAPI from './browser/microphone-api.js';

const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
const mic = new MicrophoneAPI(stream);

Quick examples

const profile = await mic.calibrate(3);
console.log('Calibration:', profile);

mic.startRealtime((data) => {
  console.log('Live frame:', data);
});

mic.stopRealtime();

Why the “Requested device not found” error appears

No mic connected Browser permission blocked Selected input unavailable

The demo now retries with simpler constraints and shows clearer fallback messages. If your browser still cannot find an input, connect a microphone or choose a different input from the browser permission dialog.

Constructor

new MicrophoneAPI(stream, config)

  • analyserFftSize (default 2048)
  • captureIntervalMs (default 50)
  • vadThreshold (default 0.03)
  • vadSilenceFrames (default 5)
  • clippingThreshold (default 0.98)
  • noiseFloorMultiplier (default 1.8)

Methods

  • calibrate(seconds) — samples ambient noise and returns a calibration profile.
  • analyzeFrame() — returns one snapshot of audio metrics.
  • startRealtime(callback) — streams metric frames continuously.
  • stopRealtime() — stops the realtime loop.
  • getDeviceInfo() — returns device metadata and permission state.
  • explain() — returns supported metric capabilities.

Live data fields

FieldDescription
inputLevelInstantaneous amplitude estimate from the time domain.
peakLevelMaximum absolute sample value in the current buffer.
rmsLevelRoot mean square energy for loudness and voice activity detection.
noiseFloorEstimated noise floor from calibration.
speechProbabilityConfidence that speech is present.
pitchEstimated dominant pitch in Hertz.
roomScoreSimple room-quality estimate from spectral content.
frequencySpectrumFFT magnitudes for the analyzer bins.
dominantFrequencyPeak frequency in the current spectrum.
